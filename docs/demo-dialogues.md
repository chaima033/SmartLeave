# Dialogues de Demonstration Chatbot

## Scenario 1: Recherche / consultation

**Utilisateur:** Quels CV sont disponibles pour le poste Laravel ?

**Chatbot:** Vous avez 2 CV correspondant au mot-clé Laravel. Le CV principal de Karim est le plus complet, et celui de Sara indique aussi PHP/MySQL.

## Scenario 2: Analyse / recommandation

**Utilisateur:** Quel candidat me conseilles-tu pour une mission backend ?

**Chatbot:** Le meilleur profil actuellement est Karim: son CV principal mentionne Laravel, API REST, MySQL et 4 ans d'experience. Il a aussi une candidature active sur une offre similaire.

## Scenario 3: Gestion d'erreur / hors contexte

**Utilisateur:** Donne-moi la météo de demain.

**Chatbot:** Je suis specialise dans les donnees de recrutement de l'application. Je peux vous aider sur les CV, offres, candidatures et profils entreprise.

## Scenario 4: Donnees metier

**Utilisateur:** Quelles offres sont encore publiees ?

**Chatbot:** 3 offres sont encore publiees: Developpeur Laravel, Integrateur Web et Analyste Data. Voulez-vous le detail d'une offre ?
